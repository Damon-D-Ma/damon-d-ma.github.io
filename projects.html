Duber Paint

Duber Paint is a locally-hosted online whiteboard application that allows users to create and join rooms to collaboratively draw on the same canvas. Users are additionally able to export and save their work as an image for later.

Made with: Python, Pygame, GitHub

Source code available on GitHub

Boggle Party

A final course project for the CSC207: Introduction to software design course. Boggle Party is a twist to the classic Boggle board game. Players can choose to either play alone or with a friend in turn-based, timed gameplay. Each player is initially given one minute and a unique randomly-generated board to guess a word. Guessing a correct word resets the time left in their turn, while guessing an incorrect or duplicate word will automatically end it.

Made with: Python, Pygame, GitHub, Agile Scrum Development Process

The source code to this project is currently private at the request of the University of Toronto. To view the source code, please kindly request it via email at damon.ma@mail.utoronto.ca.

Tuesday

A Google Chrome extension that allows users to highlight dates on a webpage and adds it as an event to their Google Calendar. Tuesday is able to recognize dates in numerous formats to maximize convenience and works on all seven days of the week despite its name.

Made with: JavaScript, RegEx, HTML, CSS

Source code available on GitHub

Tetris

A course project assigned in CSC207: Introduction to Software Design at the University of Toronto. This version of the well-known video game Tetris allows players to save and load previous games so that they can be continued at later times. Additionally, players are able to adjust the speed of the game to adjust difficulty.

Made with: Java, JavaFX, Git

The source code to this project is currently private at the request of the University of Toronto. For inquiries regarding this project, please kindly request it via email at damon.ma@mail.utoronto.ca.
